 PS：实验五 JSP JavaBean 技术相关

#### 文件结构：

```bash
exp05/
├── src/main/java/com/lyon/exp05/
│   └── userBean.java              # JavaBean 数据模型
├── src/main/webapp/
│   ├── index.jsp                  # 首页
│   ├── login.jsp                  # 登录页面
│   ├── register.jsp               # 注册页面
│   ├── treatLogin.jsp             # 登录处理页面
│   ├── treatRegister.jsp          # 注册处理页面
│   ├── successLogin.jsp           # 登录成功页面
│   ├── failLogin.jsp              # 登录失败页面
│   ├── successRegister.jsp        # 注册成功页面
│   └── failRegister.jsp           # 注册失败页面
└── pom.xml                        # Maven 配置文件
```

注意：Java 代码统一存储在 java 文件夹下

---

#### useBean 标签：

语法格式：

```jsp
// id 为 Bean 实例的唯一标识符
// class 为完整的类名
// scope 表示作用域范围
// property="*" 自动匹配表单字段名与 Bean 属性名

<jsp:useBean id="user" class="com.lyon.exp05.userBean" scope="page"/>
<jsp:setProperty name="user" property="*"/>
```

前后对比:

```jsp
传统方法实例：
<%
    String username = request.getParameter("username");
    String password = request.getParameter("password");
    userBean user = new userBean();
    user.setName(username);
    user.setPassword(password);
%>

useBean 方法：
<jsp:useBean id="user" class="com.lyon.exp05.userBean" scope="page"/>
<jsp:setProperty name="user" property="*"/>
```
