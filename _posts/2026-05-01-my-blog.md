# git相关学习
## 学习的知识：

如何将写好的blog上传到GitHub，通过`git`

```bash
mv 博客文件地址 ./_posts/
将博客文件添加到目录

git status
查看变更

git add 博客文件地址
将文件添加到暂存区

git commit -m "添加 博客文章"
提交变更

git push origin main
推送到 Github
```
