#### 函数闭包

闭包：在嵌套函数中，内部函数引用了外部函数的变量(不为全局变量)，并且外部函数返回了内部函数

```py
# 嵌套函数：函数内部定义另一个函数
# 引用外部变量：内部函数使用了外部函数的变量（非全局）
# 外部函数返回内部函数：将内部函数对象作为返回值
def outer(x):
    def inner(y):
        return x + y   # 引用外部变量 x
    return inner

closure = outer(10)   # closure 是一个闭包
print(closure(5))     # 15
```

#### 正则表达式

| 函数                            | 功能                         | 返回值                        | 说明                                       |
| ------------------------------- | ---------------------------- | ----------------------------- | ------------------------------------------ |
| `re.match(pattern, string)`     | 从字符串**开头**匹配         | 匹配对象 / `None`             | 只匹配开头，不匹配也返回 None              |
| `re.search(pattern, string)`    | 在字符串中**搜索**第一个匹配 | 匹配对象 / `None`             | 只要包含即匹配                             |
| `re.findall(pattern, string)`   | 查找所有匹配                 | **列表**（每组是字符串/元组） | 无分组时返回字符串列表；有分组返回元组列表 |
| `re.finditer(pattern, string)`  | 查找所有匹配（迭代器）       | 迭代器（每个元素为匹配对象）  | 内存更优                                   |
| `re.sub(pattern, repl, string)` | 替换所有匹配                 | 新字符串                      | `repl` 可以是字符串或函数                  |
| `re.split(pattern, string)`     | 按模式分割字符串             | 列表                          | 类似 `str.split()`，但更灵活               |
| `re.compile(pattern)`           | 预编译正则表达式             | 正则对象                      | 多次使用时可提高效率                       |

| 序列 | 等价            | 含义                                 |
| ---- | --------------- | ------------------------------------ |
| `\d` | `[0-9]`         | 匹配数字                             |
| `\D` | `[^0-9]`        | 匹配非数字                           |
| `\w` | `[a-zA-Z0-9_]`  | 匹配单词字符（字母、数字、下划线）   |
| `\W` | `[^\w]`         | 匹配非单词字符                       |
| `\s` | `[ \t\n\r\f\v]` | 匹配空白字符（空格、制表符、换行等） |
| `\S` | `[^\s]`         | 匹配非空白字符                       |
| `\b` | 单词边界        | 匹配单词开头或结尾，如 `\bword\b`    |
| `\B` | 非单词边界      | 与 `\b` 相反                         |

#### 异常捕获顺序

```py
try:
    result = 1 / 0          # 抛出 ZeroDivisionError
except ValueError:
    print("捕获到 ValueError")
except ZeroDivisionError:
    print("捕获到 ZeroDivisionError")   # 这一分支会执行
except Exception:
    print("捕获到通用异常")
```



当 try 块中发生异常时，依次检查每个 except 子句，找到第一个能够匹配该异常类型的 except 分支，然后进入该分支执行，后续所有的 except 分支将被跳过

* else  仅在 try 块无异常时执行
* finally 无论是否捕获异常都会执行

#### 自定义装饰器

```python
import time
def timer(func):
    # 定义包装函数
    def wrapper(*args, **kwargs):
        # 记录函数执行前的当前时间
        start = time.time()
        
        # 调用原函数，并获取返回值
        # *args 负责接收所有的位置参数
        # **kwargs 负责接收所有的关键字参数
        result = func(*args, **kwargs)
        
        # 记录函数执行后的当前时间
        end = time.time()
        print(f"{func.__name__} 耗时 {end - start:.4f} 秒")
        
		# 将原函数执行的结果返回，确保原函数功能不变
        return result
    return wrapper

@timer  # 为函数添加装饰器
def slow_task(n):
    sum([i**2 for i in range(n)])

# slow_task = timer(slow_task)
slow_task(10 ** 5)
```

#### 面向对象编程

```python
class Book:
    total_books = 0
    def __init__(self, title: str, author: str, year: int):
        self.title = title
        self.author = author
        self.year = year
        Book.total_books += 1
    def get_info(self) -> str:
        return f"{self.title} 作者：{self.author}，{self.year}年出版"
    # 类方法，类似静态方法
    # <=> Book.total_books
    @classmethod
    def get_total(cls) -> int:
        return cls.total_books
    
# 方法的调用：
print(Book.get_total())  # 输出: 0
book1 = Book("Python基础", "张三", 2024)
book2 = Book("算法导论", "李四", 2025)

print(book1.get_info())  # 输出: Python基础 作者：张三，2024年出版
print(book2.get_info())  # 输出: 算法导论 作者：李四，2025年出版

print(Book.get_total())  # 输出: 2
```
