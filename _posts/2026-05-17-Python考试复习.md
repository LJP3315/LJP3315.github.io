#### 集合的特性

* 无序性，唯一性，可变性
* 常用方法：
  1. add(elem)：添加一个元素
  2. remove(elem)：删除指定元素

#### 装饰器基础

```py
def decorator(func):
    def wrapper():
        print("Before")
        func()
        print("After")
    return wrapper

@decorator
def hello():
    print("Hello")

hello()
# 输出：
# Before
# Hello
# After
```

#### 变量作用域

```python
# Local: 当前函数内部
# Enclosing: 外层函数
# Global: 全局变量，模块最外层
# nonlocal: 在嵌套函数中修改外层函数
x = "global"          # Global
def outer():
    x = "outer"       # Enclosing
    def inner():
        x = "inner"   # Local
        print(x)      # 输出 "inner"
    inner()
outer()
```

#### 字符串分割、切片、逆序

###### 分割：

`.split()` 方法：`s.split(",")` 按逗号分割，返回列表类型

##### 字符串切片：

 `s[start:stop:step]` 返回新字符串，原字符串不变，左开右闭

```py
s = "Hello World"
print(s[6:])        # "World"
print(s[::2])       # "HloWrd"
print(s[-5:])       # "World"
print(s[-1::-1])    # "dlroW olleH"  （与 s[::-1] 等价）
print(s[2:5])       # "llo"
```

##### 字符串逆序：

切片逆序：s[::-1]

#### 列表推导式语法：

`[x**2 for x in range(5)]` → `[0, 1, 4, 9, 16]`

`[x for x in range(10) if x % 2 == 0]` → `[0, 2, 4, 6, 8]`

`[x*y for x in [1,2] for y in [3,4]]` → `[3, 4, 6, 8]`

`[x if x%2==0 else -x for x in range(5)]` → `[0, -1, 2, -3, 4]`

#### 函数闭包

闭包：在嵌套函数中，内部函数引用了外部函数的变量(不为全局变量)，并且外部函数返回了内部函数

```py
# 嵌套函数：函数内部定义另一个函数
# 引用外部变量：内部函数使用了外部函数的变量（非全局）
# 外部函数返回内部函数：将内部函数对象作为返回值
def outer(x):
    def inner(y):
        return x + y   # 引用外部变量 x
    return inner

closure = outer(10)   # closure 是一个闭包
print(closure(5))     # 15
```
