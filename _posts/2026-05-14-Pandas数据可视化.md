#### 第三部分 - 绘制柱状图

```py
plt.figure(figsize=(10, 5))
bins = list(range(0, 101, 10))
labels = [f'{i} - {i+9}' for i in range(0, 100, 10)]
df['分数段'] = pd.cut(df['成绩'], bins=bins, labels=labels, right=False, include_lowest=True)

counts = df['分数段'].value_counts().sort_index()
counts.plot(kind='bar', color='skyblue', edgecolor='black')
plt.title('成绩分布柱状图')
plt.xlabel('分数段')
plt.ylabel('人数')
plt.xticks(rotation=45)
plt.tight_layout()
plt.show()
```

##### 代码分析

```py	
plt.figure(figsize=(10, 5))
bins = list(range(0, 101, 10))
labels = [f'{i} - {i+9}' for i in range(0, 100, 10)]
```

* 创建新的图形画布，设置宽为10英寸。高为5英寸
* 然后设置分组的边界点，11个点，表示10个区间
* 为每个区间创建字符串标签

---

```py
df['分数段'] = pd.cut(df['成绩'], bins=bins, labels=labels, right=False, include_lowest=True)
```

* 调用 pd.cut() 方法：将连续数值分割为离散区间
* 分割成绩列，bins 表示 边界列表，labels 表示 每个区间对应的标签，
  right=False 表示左开右闭，include_lowest=True 表示 将第一个区间的左边界包含

---

```py	
counts = df['分数段'].value_counts().sort_index()
```

* value_counts() : 计算每个标签出现的次数
* sort_index() : 按照分数段顺序显示

---

```py	
counts.plot(kind='bar', color='skyblue', edgecolor='black')
plt.title('成绩分布柱状图（Pandas 绘制）')
plt.xlabel('分数段')
plt.ylabel('人数')
plt.xticks(rotation=45)
plt.tight_layout()
plt.show()
```

* counts.plot(kind='bar') : 指定 kind='bar' 生成柱状图
* 设置标题，设置x, y轴的标签
* 旋转 x 轴标签，防止重叠
* 自动调整边距，使文字完整显示

#### 第四部分 - 绘制折线图

```py
df_sorted = df.sort_values('成绩', ascending=False).reset_index(drop=True)
df_sorted['学号'] = df_sorted['学号'].astype(str)
plt.figure(figsize=(12, 6))
plt.plot(df_sorted['学号'], df_sorted['成绩'],
         marker='o',
         linestyle='-',
         color='coral',
         linewidth=2)
plt.title('班级成绩分布图（按成绩从高到低排列）')
plt.xlabel('学号')
plt.ylabel('成绩')
plt.xticks(rotation=90, fontsize=8)

plt.grid(axis='y', linestyle='--', alpha=0.7)
plt.tight_layout()
plt.show()
```

##### 代码分析

```py
df_sorted = df.sort_values('成绩', ascending=False).reset_index(drop=True)
df_sorted['学号'] = df_sorted['学号'].astype(str)
```

* 对成绩进行降序排列
* 将学号转换为字符串类型，将学号作为离散的标签

---

```py
plt.figure(figsize=(12, 6))
plt.plot(df_sorted['学号'], df_sorted['成绩'],
         marker='o',
         linestyle='-',
         color='coral',
         linewidth=2)
```

* 创建新的图形窗口
* 设置x轴为学号，y轴为成绩
* 数据点设置为圆形，线型为实线，线宽为2

---

```py
plt.title('班级成绩分布图（按成绩从高到低排列）')
plt.xlabel('学号')
plt.ylabel('成绩')
plt.xticks(rotation=90)
plt.grid(axis='y', linestyle='--', alpha=0.7)
plt.tight_layout()
plt.show()
```

* 设置标题，x轴标签与y轴标签
* 设置x轴标签的旋转，避免重叠
* 设置网格线，只设置水平线，线型为虚线，透明度为0.7
