#### 步骤2. 将字典转换为DataFrame

```py
data = {
    '姓名': ['张三', '李四', '王五', '赵六', '陈七'],
    '性别': ['男', '女', '男', '女', '男'],
    '年龄': [20, 21, 19, 20, 22],
    '语文': [88, 92, 76, 85, 90],
    '数学': [95, 78, 82, 91, 85],
    '英语': [90, 88, 79, 86, 92]
}

df = pd.DataFrame(data)
```

---

```py

df.describe()
```

#### 步骤4. 按照条件筛选数据

```python
# 筛选语文 >= 85 分的学生
cond1 = df['语文'] >= 85
df_test1 = df[cond1]

# 筛选数学 > 90 分的男学生
cond2 = (df['性别' == '男']) & (df['数学'] > 90)
df_test2 = df[cond2]
```

注意: 每个条件都需要使用括号包裹，条件使用 `&`, `|` 进行连接

---

#### 步骤5. 新增列

```python
df['总分'] = df['语文'] + df['数学'] + df['英语']
df['平均分'] = df['总分'] / 3
```

---

#### 步骤6. 排序

```py
# 从高到低
df_sorted = df.sort_values(by='总分', ascending=False)
print(df_sorted)

# 重置索引
df_sorted = df.sort_values(by='平均分', ascending=False).reset_index(drop=True)
print(df_sorted)
```

使用 sort_value() 方法

* 参数 by : 指定按照哪些列进行排序（列名）
* 参数 ascending : 指定排序顺序（bool值或者bool值列表） 
  * True 表示升序，默认

使用 reset_index() 方法用于 重置索引行

* 参数 drop=False : 默认，将原来的索引保留成一列
* 参数 drop=True : 生成新的整数索引
